# Planning Calendar

## What changed

You can now plan workouts on a FullCalendar-based surface with:

- Week view and Month view
- Add workout (title + start/end)
- Move workout by drag/drop
- Remove workout from calendar
- Keyboard-accessible move/remove controls in the workout list

## How to use

1. Enter workout title, start, and end.
2. Select **Add workout**.
3. Use **Week view** or **Month view**.
4. Move or remove workouts as needed.
5. Confirm updates in the **Latest audit recompute payload** section.

## Accessibility

Core planning operations are also available through keyboard-focusable controls, not only pointer drag/drop.

## Scope

This feature is for web planning in v1.
